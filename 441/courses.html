Our Courses

Example lesson plan: Build a static web page Teaching Objectives: This course is designed to equip students with the basic skills of building static web pages, including HTML, CSS, and basic web page layout. Teaching content HTML Basics: Students are taught the basic tags and attributes of HTML to enable them to write basic web page structures. CSS Basics: An introduction to CSS selectors, style properties, and box models to enable students to add styles and layouts to web pages. Common layouts: Explain common types of page layouts, such as flowing, fixed, and responsive. Images and Media: Describes how to embed images and media files in web pages, and how to set up hyperlinks. Page Optimization: Explains how to optimize web page loading speed and accessibility, as well as how to do simple SEO optimization. Teaching methods Theoretical explanation: The teacher will explain each knowledge point in detail and give corresponding examples and exercises. Hands-on: Students follow the teacher's explanation in class to carry out practical operations and gradually master what they have learned. Class discussion: Encourage students to ask questions and suggestions to deepen their understanding of knowledge points through discussion. Group Collaboration: Students work in groups to work on a project and work together to complete a complete static web project. Methods of assessment Classroom exercises: Observe students' actual operations in class and assess their mastery of knowledge points. Project Evaluation: Students are required to submit a complete static web project to evaluate their comprehensive ability and creativity. Group Discussion Performance: Evaluate students' participation and contribution in group discussions, and understand their understanding of knowledge points. Homework: Students are asked to complete relevant practice questions and readings on time to reinforce what they have learned. Teaching features Practice-oriented: This course emphasizes students' practical ability, and enables students to truly master the skills they have learned through a lot of practical exercises. Group Cooperation: Through group cooperation project exercises, students develop teamwork and communication skills. Personalized guidance: Teachers provide personalized guidance based on students' needs and problems to help students solve their confusion in learning.
